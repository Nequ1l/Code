# Программа мониторинга пиковой нагрузки портала

## Описание
Программа предназначена для суммирования временных интервалов работы портала и вычисления общего времени работы.  
Пользователь вводит количество интервалов **N** (10 ≤ N ≤ 30), после чего вводит значения каждого временного промежутка.  
По завершении ввода программа выводит итоговое суммарное время.

## Алгоритм работы
1. Ввести число **N** — количество интервалов (10 ≤ N ≤ 30).
2. Проверить корректность диапазона N.
3. Инициализировать переменную `total = 0`.
4. Повторить N раз:
   - считать значение очередного интервала времени,
   - прибавить его к переменной `total`.
5. Вывести итоговое значение `total` — суммарное время работы портала.
6. Завершить выполнение программы.
